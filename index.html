Cómo instalar la esfera

Guía rápida para usar lo exportado desde este editor.

1. Exporta

Pulsa exportar para descargar un ZIP con `app.json`, `app.js`, `watchface/index.js` y assets mínimos.

2. Crea el proyecto

Descomprime el ZIP, entra en la carpeta del proyecto y ejecuta `zeus build -t default`.

3. Sincroniza

El `.zab` aparecerá en `dist/`; después puedes probarlo o instalarlo desde las herramientas de Zepp.

El ZIP exportado es un proyecto para compilar. Dentro también va `amazfit-editor-design.json` para reabrir el diseño en este editor si necesitas seguir modificándolo.
